About the App

We wanted to build a challenging react app and we knew a game would be something complex, we built Jeopardy because well we can… and mostly because of Kevin. Ideally we wanted to build something that didn’t require animation and given the short Time Jeopardy seem like a good idea. It also made perfect sense because it allowed us to fulfill the requirements of having a database to store all the questions.

Technolgies used

For our technologies we used Amazon web services, utilizing their built in services we were able to build a simple login and logout function. While building the app we also discovered a great library called Jservice.io, which basically held all the questions that were ever used in jeopardy and the right answ ers associated to it.

Challenges we faced

  • underwhelming computer
  • Getting everyone set up on AWS
  • Using Mongodb instead of mySQL
  • Passing data from Jservice.io to our front end.
  • validating Answers received from Jservice.io
  • Updating state and front end
  • working with looped API
